# Educational-practice---task-3
Case task #3
Кейс-задача № 3
Создайте простой слайдер изображений.
1) Создайте интерфейс с помощью HTML и CSS, состоящий из области отображения изображений и кнопок "вперед" и "назад";
2) Стилизуйте интерфейс, чтобы он был привлекательным: добавьте рамки, тени, выберите подходящие цвета и шрифты;
3) Реализуйте функционал смены изображений с помощью ванильного JavaScript и слушателей событий. При нажатии на кнопку "вперед" должно отображаться следующее изображение, при нажатии на кнопку "назад" - предыдущее;
4) Добавьте функционал, который будет обеспечивать зацикливание слайдера, то есть после последнего изображения снова отображается первое, а перед первым - последнее;
5)	Дополните интерфейс возможностью отображения текущего номера изображения (например, "Изображение 1 из 5") для удобства пользователя.
